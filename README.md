# Personal Portfolio

A highly interactive and immersive personal portfolio website built with React, Vite, and Framer Motion.

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```

## Features

-   **Immersive Animations**: Powered by Framer Motion.
-   **Responsive Design**: Works on all devices.
-   **Dark Mode**: Sleek dark theme with neon accents.
-   **Interactive Components**: Hover effects, scroll reveals, and more.

## Customization

-   Update `src/components/*.jsx` with your own content.
-   Modify `src/index.css` to change the color scheme.
